QML中的基本类型list

QML对象的列表。

list类型引用QML对象的列表。

list值可以通过类似于JavaScript数组的方式访问:

  • 值是使用带逗号分隔值的[] 方括号语法分配的
  • length 属性提供列表中的项数
  • 使用[index]语法访问列表中的值

值可以通过使用push 方法动态地添加到列表中，就像它是一个JavaScript数组一样

list只能存储QML对象，不能包含任何基本类型值。(要在列表中存储基本类型，请使用var类型。)

与C++集成时，请注意，从C++传递到QML的任何QQmlListProperty 值都会自动转换为list 值，反之亦然。

使用list类型

例如，Item 类型有一个states 列表类型的属性，可以分配给并使用如下:


  import QtQuick 2.0

  Item {
      width: 100; height: 100

      states: [
          State { name: "activated" },
          State { name: "deactivated" }
      ]

      Component.onCompleted: {
          console.log("Name of first state:", states[0].name)
          for (var i = 0; i < states.length; i++)
              console.log("state", i, states[i].name)
      }
  }

已定义的State对象将按照定义它们的顺序添加到states列表中。

如果列表只包含一个对象，可以省略方括号:


  import QtQuick 2.0

  Item {
      width: 100; height: 100
      states: State { name: "activated" }
  }

注意，对象一旦创建，就不能单独添加或从list中删除;要修改列表的内容，必须将其重新分配到一个新list中。

注意:list 类型不建议作为自定义属性的类型。相反，应该使用var 类型，因为可以在QML中更灵活地操作var 类型存储的列表。

This basic type is provided by the QML language.
