QJSEngine的过时成员

下列的QJSEngine类的成员已经过时。提供它们是为了让旧的源代码继续工作。我们强烈建议不要在新代码中使用它们。

公用函数

(obsolete) dumpObjectInfo()
(obsolete) dumpObjectTree()
(obsolete) void installTranslatorFunctions(const QJSValue &object = QJSValue())

成员函数文档

void QJSEngine::installTranslatorFunctions(const QJSValue &object = QJSValue())

这个函数已经过时了。提供它是为了保持旧的源代码工作。我们强烈建议不要在新代码中使用它。

在给定 object上安装转换器函数，如果没有指定对象，则在全局对象上安装转换器函数。

脚本翻译函数与C++翻译函数的关系如下表所示:

Script FunctionCorresponding C++ Function
qsTr()QObject::tr()
QT_TR_NOOP()QT_TR_NOOP()
qsTranslate()QCoreApplication::translate()
QT_TRANSLATE_NOOP()QT_TRANSLATE_NOOP()
qsTrId()qtTrId()
QT_TRID_NOOP()QT_TRID_NOOP()

它还向字符串原型添加了一个arg()方法。

这个函数是在Qt 5.4中引入的。
