QML的基本数据类型enumeration

命名枚举值。

enumeration类型引用命名的枚举值。

每个命名值都可以称为<Type>.<value>.例如，Text类型有一个 AlignRight枚举值:


  Text { horizontalAlignment: Text.AlignRight }

(为了向后兼容，枚举值也可以指定为字符串，例如. "AlignRight". 对于新代码，不推荐使用此形式。)

当与C++集成时，请注意，从C++传递到QML的任何enum 值都会自动转换为enumeration值，反之亦然。

这种基本类型由QML语言提供。QtQuick导入提供了一些枚举值。

使用QML中的enumeration类型

enumeration类型C++enum类型的表示形式。 无法引用QML本身中的enumeration类型;相反，当引用QML代码中的enumeration值时，可以使用int 或 var类型。

例如：


  import QtQuick 2.0

  Item {
      // refer to Text.AlignRight using an int type
      property int enumValue: textItem.horizontalAlignment

      signal valueEmitted(int someValue)

      Text {
          id: textItem
          horizontalAlignment: Text.AlignRight
      }

      // emit valueEmitted() signal, which expects an int, with Text.AlignRight
      Component.onCompleted: valueEmitted(Text.AlignRight)
  }

请参见QML基本类型 和 枚举属性。
