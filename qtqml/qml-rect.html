QML基本类型rect

具有x、y、宽度和高度属性的值。

rect 类型引用具有x, y, width 和 height属性的值。

例如，读取Item的childrenRect的rect类型的 width 和 height 的属性:


  Rectangle {
      width: childrenRect.width
      height: childrenRect.height

      Rectangle { width: 100; height: 100 }
  }

要创建rect值，请使用"x, y, width x height"形式的字符串代码：


  CustomObject { myRectProperty: "50,50,100x100" }

或者使用Qt.rect()函数：


  CustomObject { myRectProperty: Qt.rect(50, 50, 100, 100) }

rect类型还公开只读的left, right, top 和 bottom属性。它们与C++中的对应项相同。

当与C++集成时，请注意，从C++传递到QML的任何QRect 或 QRectF值都会自动转换为rect值，反之亦然。当一个rect 值传递给C++时，它会自动转换成一个QRectF值。

请参见QML基本类型。
