QML的DelegateChooser 类型

允许视图对模型中不同类型的项使用不同的委托。更多…

Import Statement: import Qt.labs.qmlmodels 1.0
  • 所有成员的列表，包括继承的成员

属性

  • choices : list<DelegateChoice>
  • role : string

详细信息

DelegateChooser 是一种特殊的Component（组件） 类型，适用于视图需要组件并将其用作委托的场景。 DelegateChooser 装了一组DelegateChoice。 这些选项用于确定将为模型中的每个项实例化的委托。选项的选择是基于模型项对role（角色）的值，也基于索引执行的。

注意:这种类型只适用于TableView 和任何基于。DelegateModel的视图。内部不基于DelegateModel 的视图(包括用户定义的视图)需要显式地支持这种类型的组件，以使其像描述的那样工作。

请参见DelegateChoice。

属性文档

[default] choices : list<DelegateChoice>

选择器的DelegateChoices列表。

该列表被视为有序列表，其中将使用第一个要匹配的DelegateChoice作为视图。

通常不需要引用choicese属性，因为它是DelegateChooser 默认属性，因此所有子项目都会被自动分配这个属性。


role : string

此属性包含用于确定给定模型项的委托的角色。

请参见DelegateChoice。
