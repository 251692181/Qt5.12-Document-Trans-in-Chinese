QML中的Qt QML类型

Qt QML模块提供了可以与QML语言一起使用的各种便利类型的定义和实现，包括一些基本的QML类型，它们可以为QML语言的进一步扩展提供基础。QtObject 和 Componentt对象类型是非可视的，并且为QML扩展提供了构建块。

导入QtQml

QtQml模块提供的类型仅在QML文档中可用，如果该文档导入QtQml 名称空间(或者如果该文档导入QtQuick名称空间，如下所述)。

QtQml模块的当前版本为2.12版本，可以通过以下语句导入:


  import QtQml 2.12

大多数客户端永远不需要使用QtQml导入，因为所有类型都由QtQuick命名空间提供，可以导入如下:


  import QtQuick 2.12

有关QtQuick名称空间及其提供给QML应用程序开发人员的内容的更多信息，请参阅Qt Quick。

用于创建列表和模型的QML类型(如ListModel 和 ListElement)被移动到子模块QtQml.Models上。有关更多信息，请参阅Qt QML模块。

下面的类型文档同样适用于由Qt Quick模块提供的同名类型，因为它们实际上是相同的。

基本类型

提供了以下QML基本类型:

date

日期值

point

具有x和y属性的值

rect

具有x、y、width和height属性的值

size

具有width和height属性的值

Object类型

提供了以下QML object类型：

Binding

支持任意属性绑定的创建

Component

封装QML组件定义

Connections

描述到信号的广义连接

Date

提供日期功能

Instantiator

动态创建对象

Locale

提供特定于区域设置的属性和格式化数据

LoggingCategory

在QML中定义日志记录类别

Number

该对象表示一个数值

Qt

提供一个全局对象，其中包含有用的枚举和Qt中的函数

QtObject

一个基本的QML类型

String

对象表示字符串值

Timer

以指定的间隔触发处理程序
