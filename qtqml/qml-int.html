int QML Basic Type

一个整数，如. 0, 10, 或 -20.

int类型指的是一个整数，例如. 0, 10, 或 -20.

可能的int 值范围在-2000000000到2000000000之间，尽管大多数类型只接受较小的范围(他们在文档中提到过)。

例子


  Item { width: 100; height: 200 }

这种基本类型由QML语言提供。

参见QML基本类型。.
