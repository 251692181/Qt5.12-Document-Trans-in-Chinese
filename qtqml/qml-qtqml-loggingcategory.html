QML的LoggingCategory类型

在QML中定义日志记录类别。更多…

Import Statement: import QtQml 2.12
Since: Qt 5.8
  • 所有成员的列表，包括继承的成员

属性

  • defaultLogLevel : enumeration
  • name : string

详细描述

日志类别可以作为第一个参数传递给console.log() 和 friends。 如果提供了日志名称级别，提供给日志程序LoggingCategory的名称将用作日志类别，否则将使用默认日志类别。


  import QtQuick 2.8

  Item {
      LoggingCategory {
          id: category
          name: "com.qt.category"
          defaultLogLevel: LoggingCategory.Warning
      }

      Component.onCompleted: {
        console.log(category, "message");
      }
  }

注意:由于创建对象的开销很大，因此建议将所需的LoggingCategory 义放到一个单例中，并在需要的地方导入它。

请参见QLoggingCategory。

属性文档

defaultLogLevel : enumeration

保存日志类别的默认日志级别。默认情况下，它是用LoggingCategory创建的。调试日志级别。

注意: 在声明LoggingCategory时需要设置此属性，以后不能更改。

这个属性是在Qt 5.12中引入的。


name : string

保存日志类别的名称。

注意：在声明LoggingCategory时需要设置此属性，以后不能更改。

请参见 QLoggingCategory::categoryName()。
