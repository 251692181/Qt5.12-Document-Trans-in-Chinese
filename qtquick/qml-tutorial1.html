QML Tutorial 1 - Basic Types

第一个程序是一个非常简单的"Hello world" 示例，介绍了一些基本的QML概念。下图是这个程序的截图。

以下是应用程序的QML代码:


  import QtQuick 2.0

  Rectangle {
      id: page
      width: 320; height: 480
      color: "lightgray"

      Text {
          id: helloText
          text: "Hello world!"
          y: 30
          anchors.horizontalCenter: page.horizontalCenter
          font.pointSize: 24; font.bold: true
      }
  }

Walkthrough（预排工作）

Import（导入）

首先，我们需要导入本例所需的类型。大多数QML文件将导入Qt自带的内置QML类型(例如 Rectangle（矩形）, Image（图片）, ...) 使用:


  import QtQuick 2.0

Rectangle Type（矩形类型）


  Rectangle {
      id: page
      width: 320; height: 480
      color: "lightgray"

我们声明一个Rectangle类型的根对象。它是您可以用来在QML中创建应用程序的基本构建块之一。 我们给它一个 id，以便以后能够引用它。 在本例中，我们称其为 "page（页面）"。我们还设置了width（宽）, height（高） 及 color（颜色） 属性。Rectangle矩形类型包含许多其他属性(如x 和 y)，但这些属性保留默认值

Text Type（文本类型）


      Text {
          id: helloText
          text: "Hello world!"
          y: 30
          anchors.horizontalCenter: page.horizontalCenter
          font.pointSize: 24; font.bold: true
      }

我们添加一个Text(文本类型) 作为根矩形类型的子类型，它显示文本'Hello world!'.

y属性用于将文本垂直放置在距其父元素顶部30像素处。

anchors.horizontalCenter属性是指定类型的水平中心。在本例中，我们指定文本类型在page（页面） 元素中水平居中位置（参见 Anchor-Based Layout（基于锚的布局））。

font.pointSize 和 font.bold属性与字体相关，并使用点表示法。

查看示例

要查看创建的内容，可以运行qmlscene工具(位于bin目录中)，文件名作为第一个参数。例如，要从安装位置运行提供的完整教程1示例，您可以输入:


  qmlscene tutorials/helloworld/tutorial1.qml
