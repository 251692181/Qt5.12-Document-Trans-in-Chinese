Rectangle QML Type

用可选边框绘制填充矩形。更多…

详细描述

矩形项用于用纯色或渐变填充一个矩形区域，并可能提供矩形边框。

外观

每个矩形项要么使用使用color属性指定的实色填充， 要么使用渐变类型定义并使用gradient（渐变）属性设置渐变。如果同时指定了颜色和渐变，则使用渐变。

你可以通过设置border.color和border.width属性来设置矩形的边框和宽度。将颜色设置为"transparent" （透明），以绘制没有填充颜色的边框

还可以使用radius属性创建圆角矩形。由于这会将将矩形的每个角设置为曲边（会产生锯齿），因此可以适当地设置Item::antialiasing（反锯齿）属性来改进它的外观。

使用示例

下面的示例显示了一些常用属性对矩形项的影响，在本例中，矩形项用于创建正方形：


  import QtQuick 2.0

  Rectangle {
      width: 100
      height: 100
      color: "red"
      border.color: "black"
      border.width: 5
      radius: 10
  }

性能性质

使用Item::antialiasing（反锯齿）属性可以改善圆角矩形的外观，但要牺牲显示性能。您应该考虑为移动中的矩形不设置此属性，并只在它们静止时才设置。

请参阅Image（图片）。

性质性能文档

antialiasing : bool

用于决定矩形是否应该使用反锯齿。参见Antialiasing（反锯齿），其提供关于此属性的性能影响的信息。

默认值是true，意味着矩形圆角radius，否则为false，这意味着其他情况。


border group

border.width : int

border.color : color

用于绘制矩形边框的宽度和颜色。

宽度为1创建一条细线。如果没有指定线条样式，则使用宽度为0或透明颜色绘制。

注意：如果使用锚点，矩形边框的宽度不影响矩形本身的几何形状或相对于其他项的位置。（既矩形边框是附加的，不参与矩形大小和位置计算）


color : color

此属性指定用于填充矩形的颜色。

默认的颜色是白色。

下面的例子显示了使用十六进制和命名颜色符号指定的颜色绘制矩形:


  Rectangle {
      color: "#00B000"
      width: 80; height: 80
  }

  Rectangle {
      color: "steelblue"
      y: 100; width: 80; height: 80
  }

如果指定了渐变属性和其颜色，则使用渐变。

参见gradient（渐变）。


gradient : Gradient

用来填充矩形的渐变。

该属性允许构造简单的垂直渐变。其他垂直可以通过向矩形添加旋转来形成。


  Rectangle {
      y: 0; width: 80; height: 80
      color: "lightsteelblue"
  }

  Rectangle {
      y: 100; width: 80; height: 80
      gradient: Gradient {
          GradientStop { position: 0.0; color: "lightsteelblue" }
          GradientStop { position: 1.0; color: "blue" }
      }
  }

  Rectangle {
      y: 200; width: 80; height: 80
      rotation: 90
      gradient: Gradient {
          GradientStop { position: 0.0; color: "lightsteelblue" }
          GradientStop { position: 1.0; color: "blue" }
      }
  }

如果指定了渐变和其颜色，则使用渐变。

参见Gradient（渐变）和color（颜色）。


radius : real

此属性包含用于绘制圆角矩形的角半径。

如果半径非零，则矩形将被绘制为圆角矩形，否则将被绘制为普通矩形。四个角的半径相同;目前没有办法为不同的角指定不同的半径。
